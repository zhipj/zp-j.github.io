Adjust Raspberry Pi Camera focus and explain how it works

May 28, 2020

Device: Raspberry Pi 3 B+
Camera: Raspberry Pi Camera v2.1

I tried to use a Raspberry Pi to shoot a timelapse video, to record the growth of my baby tangerine.

However, after checking the first pilot, I found that the camera was focusing on somewhere else, not the plant. For example, the floor is clear while the plant is blurry. I don’t expect it to be smart enough to auto focus on an object for the price that I paid, but I’m wondering if there’s anything that we can tweak to improve it.

How to adjust Raspberry Pi Camera’s focus?

The standard Pi Camera is a fixed focus lens, it’s not like a zoom lens. However, the lens base is like a screw in a nut. You can adjust the lens focus by tightening or loosening the lens base, amazing!

There are some posts talking about it. What surprises me is people are so smart that they can find the workaround, it doesn’t look like an official solution by design.

If you look closer at the double arrow, you can use a small tweezer to rotate it.

How does it work?

To simplify the structure, the sensor is at the bottom, the lens is on top. When you rotate the lens base, the height is changed, which then changes the internal lens position so that it can focus on a different distance.

Now let’s see how different lens positions can change the focus distance.

In the above two cases, when you tighten the lens base clockwise, h decreases to h', vice versa. When the lens base moves to the left, the focus point moves too, in the same direction. That explains why we can have a shorter focus distance.

Pay Attention

  • Be gentle, don’t screw it too hard
  • When you screw anti-clockwise, don’t rotate too far, the lens base will pop out, it’s not that easy to put it back
  • If you are using a tool like knife, be care not to hurt yourself
  • Some people suggest to do it with camera unplugged, well, I did it with it connected, so maybe it’s OK
